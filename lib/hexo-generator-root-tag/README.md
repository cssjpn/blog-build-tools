# hexo-generator-root-tag

Tag generator for [Hexo](https://hexo.io/).

## Installation

``` bash
$ npm install jpazureid/hexo-generator-root-tag --save
```

## Options

``` yaml
root_tag_generator:
  root_tag_names:
    - Azure AD
    - AAD Connect
    - AD FS
    - 情報採取
  sub_tag_limit: 10
  sub_tag_sort: 'name' # 未実装
```
